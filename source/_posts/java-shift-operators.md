java中位移运算符可参考 C 中位移运算，但 Java 中较 C 多出一种无视符号位的位移

```java
3>>>1
```

意思为，忽略符号位进行位移

会忽视符号位的影响（如负数位移补码中补位的是1）

Java中，对于位移负数位有特别算法

例如：

```java
3>>>-2
```

其实等价于

```java
3>>>30
```

位移负数，取位移数后五位（long，double取6位），二值化，忽略符号位，

```
数字2二值化： 0000  0010  =>  2
取负数 补码： 1111  1110  => -2
取出后五位：  111\|11110  => -2
十进制化：         11110  => 30
```

做这种转换后，得出移动值
