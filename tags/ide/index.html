配置、插件 同步插件 Settings Sync

最近发现了一个好东西，可以借助 github gist 给 vscode 提供一个配置同步，这款插件名为 Settings Sync

使用十分方便，在新安装好的 vscode 上安装这款插件，登录 github 账号，并配置 gist 地址，就可以同步配置
